  - 移除 systemNotificationApi 调用，改用 conversationApi.getNotificationChannel()

  - 支持激活状态高亮和响应式设计
- ✅ 五子棋盘棋子对齐修复
  - **根因**：CSS Grid 布局引擎定位单元格 与 绝对定位 calc() 绘制网格线使用两套不同坐标系统
  - 浏览器亚像素渲染差异导致棋子始终无法精确对齐网格线交点
  - **方案**：用 CSS 伪元素 `::before`/`::after` 在每个单元格内绘制网格线
  - 网格线和棋子共享同一个单元格的 50%/50% 中心点，从根本上保证对齐
  - 移除 30 个绝对定位网格线 div 元素，简化 DOM 结构
  - 外边框保留绝对定位，z-index 高于网格线确保边缘视觉正确
- ✅ 中国象棋游戏全栈开发完成
  - **chessEngine.ts**: 完整象棋规则引擎（600行）
    - 7种棋子类型 + 完整走法规则（王/仕/相/車/馬/炮/兵）
    - 九宫限制、象不过河、马蹩脚、炮翻山吃子

    - Medium: Alpha-Beta深度3 + 走法排序优化
    - Hard: Alpha-Beta深度4 + 开局库 + 将军加分
    - 走序优化：MVV-LVA（大值吃小值优先）
  - **ChineseChessBoard.tsx**: React棋盘组件（400行）
    - 使用与Gomoku一致的CSS伪元素网格线方案，确保对齐
    - 棋子选中高亮 + 合法走法圆点/捕获环指示
    - 将军闪烁动画、最后落子标记
