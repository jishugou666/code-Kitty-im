## 一、紧急修复项

### 1.1 数据库表结构修复

**问题描述**: AI反垃圾和IP封禁功能的数据库表缺少必要字段，导致功能异常。

**修复方案**:

```sql
-- 1. 确保 ban_status 字段存在且默认值正确
ALTER TABLE `user` ADD COLUMN IF NOT EXISTS `ban_status` ENUM('active', 'banned') DEFAULT 'active';

-- 2. 确保 user_ip_log 表存在
CREATE TABLE IF NOT EXISTS `user_ip_log` (
  `id` INT PRIMARY KEY AUTO_INCREMENT,
  `user_id` INT NOT NULL,
  `ip_address` VARCHAR(45) NOT NULL,
  `login_time` TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  INDEX idx_user_id (`user_id`),
  INDEX idx_ip_address (`ip_address`)
);

-- 3. 确保 ip_ban 表存在
CREATE TABLE IF NOT EXISTS `ip_ban` (
  `id` INT PRIMARY KEY AUTO_INCREMENT,
  `ip_address` VARCHAR(45) NOT NULL,
  `ban_reason` VARCHAR(500),
  `created_at` TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  INDEX idx_ip_address (`ip_address`)
);
```

**执行时间**: 立即
**优先级**: P0 - 紧急
